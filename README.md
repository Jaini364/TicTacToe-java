# Tic Tac Toe with Minimax AI (Java)

## Description
This project is a Tic Tac Toe game built using Java Swing GUI where the player plays against an AI. 
The AI uses the Minimax algorithm to make optimal moves and is impossible to beat.

## Features
- GUI based Tic Tac Toe game
- Player vs AI
- Minimax Algorithm
- Optimal AI moves
- Game result detection
- Reset game option

## Technologies Used
- Java
- Swing
- AWT
- Object Oriented Programming
- Minimax Algorithm (Artificial Intelligence)

## How the AI Works
The Minimax algorithm evaluates all possible moves and chooses the move that maximizes the AI's chance of winning and minimizes the player's chance of winning.

## How to Run
1. Open project in any Java IDE
2. Run TicTacToe.java
3. Play against AI
